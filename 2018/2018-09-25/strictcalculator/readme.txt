To reproduce the code and execute tests use python3 and pytest.

1. I am sure, that you have Python3 installed on your machine..
2. Make your environemnt for python3 and this project:
   $ python3 -m venv __venv__
3. Activate the environemnt:
   $ . ./__venv__/bin/activate
4. You can upgrade pip..
   $ pip install --upgrade pip
5. Install PyTest
   $ pip install pytest
6. Execute the test
   $ pytest
